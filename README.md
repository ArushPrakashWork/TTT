Tic Tac Toe - Console Based Mini Game in C++

A fully interactive **console-based Tic Tac Toe game** developed in **C++** using core programming concepts such as arrays, loops, conditionals, and functions.

This project is designed to work on **Windows, Linux, and Mac**, with automatic platform detection for smooth gameplay.

---

## Features

- Two-player mode
- Custom player names
- Instant keypress input (no Enter required during gameplay)
- Dynamic board refresh system
- Win and tie detection
- Rematch option
- New game option
- Cross-platform support
- Clean console UI

---

## Concepts Used

This project demonstrates the use of:

- Arrays
- Loops
- Functions
- Conditional Statements
- Global Variables
- String Handling
- Platform Detection
- Terminal Input Handling
- Console UI Rendering

---

## How to Play

- The board contains **9 cells numbered from 1 to 9**
- Press any number key **1-9** to place your mark
- Player 1 uses **X**
- Player 2 uses **O**
- First player to align **3 marks in a row** wins
- Rows, columns, and diagonals are all valid winning combinations

---

## Controls

| Key | Action |
|------|--------|
| 1-9 | Place mark |
| 1   | Rematch |
| 2   | New Game |
| 3   | Exit |

---

## Winning Conditions

A player wins if they place 3 marks in:

- Same row
- Same column
- Same diagonal
